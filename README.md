# Email Triage Environment

This environment simulates email classification tasks.

## Tasks
- Easy: clear classification
- Medium: spam detection
- Hard: ambiguous cases

## Actions
Agent must classify emails into:
- important
- spam
- work

## Reward
1 for correct classification, 0 otherwise.
